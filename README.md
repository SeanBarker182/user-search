# Welcome to User Search!

A React application used for displaying, sorting and filtering user data.

# Start Building

Follow these steps to start building locally on the project:

1. Clone the repository to your machine
2. Install the dependencies with `$ npm install`
3. Start the server with `$ npm start`
4. Open a new terminal window and type `$ gulp watch` to compile frontend scripts and styles
